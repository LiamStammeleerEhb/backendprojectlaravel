# Empty laravel
Dit project bevat niet veel

Het maakt gebruik van Laravel breeze (login) en Filament (admin panel).

Ik als PHP programmeur was eerst sceptisch over Laravel. Waarom een framework gebruiken als ik het ook zelf kan?
Toen voerde ik Laravel breeze uit. Ineens had ik een login, register, account updates, ... allemaal in twee command lines.
Dit was toch wel even gek. Ik heb al veel tijd gestoken in zelf login systemen te maken. En hier kreeg ik er gewoon eentje cadeau

Ik denk dat ik wel later nog Laravel ga meer uitproberen. Ik had dit project iets te lang uitgesteld omdat ik geen zin had om de login enzo te maken. Maar nu heb ik een andere view gekregen op laravel.
